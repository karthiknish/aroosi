# Mobile App Assets

This directory should contain:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `splash-icon.png` - Splash screen logo (288x288 recommended)
- `favicon.png` - Web favicon (48x48)

## Adding Assets

Copy your brand assets to this directory with the correct filenames.

For Aroosi branding, use the primary red color (#B91C1C) in your assets.
